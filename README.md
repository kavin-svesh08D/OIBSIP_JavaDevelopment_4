# OIBSIP_JavaDevelopment_4
Java development internship task 4 for online examination system 

To design a Java console-based online examination system that allows users to log in securely, update profiles, attempt timed exams, and view their scores.

Steps Performed

Implemented user authentication using user ID and password

Created question management with multiple-choice questions

Added profile and password update functionality

Implemented a timed examination using Timer and TimerTask

Calculated and displayed the final score after exam completion

Used

Java

JDK

Scanner, HashMap, ArrayList, Timer

A functional Java console-based online examination system demonstrating object-oriented programming, collections, user authentication, timer-based execution, and score evaluation.
